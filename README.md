# SPARK — Autonomous Completion Intelligence System

> "From Intention to Action."

SPARK is an AI-powered productivity system that doesn't just remind you of tasks — it actively ensures they get completed. Built on Google Cloud, Vertex AI, and Gemini 2.5 Flash.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 18, TypeScript, Vite, Tailwind CSS |
| Backend | FastAPI, Python 3.13 |
| AI | Vertex AI, Gemini 2.5 Flash |
| Database | Cloud Firestore, BigQuery |
| Auth | Firebase Authentication |
| Cloud | Cloud Run, Cloud Tasks, Cloud Scheduler, Pub/Sub |
| Integrations | Google Calendar, Gmail, Docs, Drive, Tasks |
